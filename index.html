Ihy project by Ihy group

Ihy codec is a new sort of audio codec, based on the mathematical tool wavelet. This is an Epita project, made by epitean.

Finally working on the iPhone (14/03/09)

The dream become reality when this evening I've heard for the first time a ihy sound on my iPod Touch. It's working perfectly which means no lags, no bugs, etc..

The only drawback is that you need a jailbroken iPhone, which needs to be at least on 2.2 firmware. Ok so let's show you how to use it with your iPhone.

First, you need a working toolchain, and a working ocaml cross-compiler, I recommend you this, and this.
Second, clone our project, and compile it for the iPhone : 

$ git clone git://github.com/Rip-Rip/ihy
$ cd ihy
$ git checkout -b iphone origin/iphone
$ cd src/cli/
$ vim Makefile     (edit the lines CROSSBINDIR, CROSSLIBDIR and CC)
$ make

If all has worked, you should see a file named ihyconvert. This is the iPhone executable. Next, you need to put it on your iPhone, and sign it with ldid. That's it, you can now listen, compress, and uncompress a ihy file. Note that it is a console program, so no graphical tool is avalaible.
